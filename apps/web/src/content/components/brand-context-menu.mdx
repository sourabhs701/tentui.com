## Features

- Copies the primary mark or logotype as raw SVG markup.
- Keeps the menu compact with three focused actions.
- Downloads the complete SVG and PNG brand kit as a ZIP archive.

## Brand asset hosting

The default TentUI SVGs and downloadable archive are served from
`https://cdn.srb.codes/brand/`. Use `markAsset`, `logotypeAsset`, and
`brandAssetsZipPath` to point the component at your own hosted assets.

## Usage

```tsx
import { BrandContextMenu } from "@/components/brand-context-menu";
```

```tsx
<BrandContextMenu>
  <button type="button">Right-click for brand assets</button>
</BrandContextMenu>
```

Pass custom mark and logotype assets and point the component to your archive.

```tsx
const mark = {
  name: "Mark",
  path: "https://cdn.example.com/brand/mark.svg",
  filename: "acme-mark.svg",
};

const logotype = {
  name: "Logotype",
  path: "https://cdn.example.com/brand/logotype.svg",
  filename: "acme-logotype.svg",
};

<BrandContextMenu
  markAsset={mark}
  logotypeAsset={logotype}
  brandAssetsZipPath="https://cdn.example.com/brand/acme-brand-assets.zip"
>
  <button type="button">Open brand kit</button>
</BrandContextMenu>;
```
