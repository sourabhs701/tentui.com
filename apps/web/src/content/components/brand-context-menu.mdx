## Features

- Copies the primary mark or logotype as raw SVG markup.
- Keeps the menu compact with three focused actions.
- Downloads the complete SVG and PNG brand kit as a ZIP archive.

## Create the brand assets ZIP

Run this command from the project root after adding or changing files in
`public/brand`:

```bash
(cd public && zip -FS -r brand/tentui-brand-assets.zip brand \
  -x "brand/.DS_Store" "brand/tentui-brand-assets.zip")
```

The component downloads the archive from
`/brand/tentui-brand-assets.zip` by default. Use `brandAssetsZipPath` if your
archive lives elsewhere.

## Usage

```tsx
import { BrandContextMenu } from "@/components/brand-context-menu";
```

```tsx
<BrandContextMenu>
  <button type="button">Right-click for brand assets</button>
</BrandContextMenu>
```

Pass custom mark and logotype assets and point the component to your archive.

```tsx
const mark = {
  name: "Mark",
  path: "/brand/mark.svg",
  filename: "acme-mark.svg",
};

const logotype = {
  name: "Logotype",
  path: "/brand/logotype.svg",
  filename: "acme-logotype.svg",
};

<BrandContextMenu
  markAsset={mark}
  logotypeAsset={logotype}
  brandAssetsZipPath="/brand/acme-brand-assets.zip"
>
  <button type="button">Open brand kit</button>
</BrandContextMenu>;
```
