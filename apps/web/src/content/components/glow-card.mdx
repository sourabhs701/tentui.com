## Features

- Tracks fine-pointer movement with a lightly damped spring.
- Keeps content composition open instead of imposing a card layout.
- Ignores touch pointers and removes the moving glow for reduced-motion users.
- Composes external pointer handlers without hiding the built-in response.

## Usage

```tsx
import { GlowCard } from "@/components/glow-card";
```

```tsx
<GlowCard glowColor="hsl(217 91% 60%)" glowSize={140}>
  <article className="rounded-[inherit] bg-card p-6">
    Your content
  </article>
</GlowCard>
```

## Accessibility

The glow is decorative and hidden from assistive technology. Touch input does
not activate pointer tracking, and reduced-motion preferences disable the moving
effect while preserving the card content and border.
