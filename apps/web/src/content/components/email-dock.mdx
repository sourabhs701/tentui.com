## Features

- Highlights the username, domain, or full address independently.
- Activates highlights on pointer hover, keyboard focus, or button click.
- Supports links and custom accessible labels for dock actions.

## Usage

```tsx
import { EmailDock } from "@/components/email-dock";
```

```tsx
const email = "hello@example.com";

<EmailDock
  email={email}
  highlights={[
    { segment: "local" },
    { segment: "domain", href: "https://example.com", external: true },
    { segment: "full", href: `mailto:${email}` },
  ]}
/>
```

Omit `highlights` to render the default username, domain, and full-address
actions without links.

## Accessibility

The address is exposed to assistive technology as one complete email address,
while its decorative segments are hidden. Each dock action is a native button or
link with an accessible label and a visible focus ring. Keyboard focus activates
the corresponding highlight.

Provide `label` when the built-in username, domain, or email label does not
describe a custom action.
