## Usage

```tsx
import { AnimatedTabs } from "@/components/animated-tabs";
```

```tsx
const tabs = [
  { value: "overview", label: "Overview" },
  { value: "analytics", label: "Analytics" },
];

<AnimatedTabs
  tabs={tabs}
  value={value}
  onValueChange={setValue}
  aria-label="Dashboard view"
/>
```

The component supports controlled and uncontrolled state with accessible radio-group semantics. Arrow keys, Home, and End move focus and selection through the options.
