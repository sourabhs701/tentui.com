## Features

- Reports loading, success, and error states for asynchronous actions.
- Animates label changes character by character.
- Announces status changes and reduces motion when requested.

## Usage

```tsx
import { ContextfulSaveButton } from "@/components/contextful-save-button";
```

```tsx
<ContextfulSaveButton
  onClick={async () => {
    await saveChanges();
  }}
/>
```

## Accessibility

The button disables itself while an action is running and announces loading,
success, and error labels through a polite live region. Reduced-motion preferences
remove transforms and stop the loading icon's rotation.
