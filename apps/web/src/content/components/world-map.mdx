## Features

- Zooms and pans across the map.
- Highlights selected countries and reports country clicks.
- Shows country names and user counts in hover tooltips.
- Disables interaction and tooltips in static mode.

## Usage

```tsx
import { WorldMap } from "@/components/world-map";
```

```tsx
const usersByCountry = {
  "United States": 1240,
  Brazil: 580,
  India: 920,
};

const selectedCountries = new Set(Object.keys(usersByCountry));

<div className="h-[450px] w-full">
  <WorldMap
    selectedCountries={selectedCountries}
    userCounts={usersByCountry}
  />
</div>
```

## Accessibility

Country interactions are pointer-only in the current implementation. Provide an
equivalent keyboard-accessible control when country selection is essential, and
do not rely on color or hover tooltips alone to communicate state.

## Examples

### Static map

Use `static` to disable zoom, pan, country clicks, and hover tooltips.

```tsx
<div className="h-[450px] w-full">
  <WorldMap static selectedCountries={new Set(["Brazil", "India"])} />
</div>
```
