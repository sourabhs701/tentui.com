## Usage

Add `group/animated-arrow` to the parent that should trigger the animation.

```tsx
import { AnimatedArrow } from "@/components/animated-arrow";
```

```tsx
<button
  type="button"
  className="group/animated-arrow inline-flex items-center gap-2"
>
  Learn more
  <AnimatedArrow />
</button>
```
