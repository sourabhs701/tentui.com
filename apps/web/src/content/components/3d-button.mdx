## Accessibility

`ThreeDButton` renders a native `button`, preserving browser keyboard, focus, and
disabled behavior. Add an accessible label to icon-only buttons and set
`type="submit"` explicitly when submitting a form.

Reduced-motion preferences remove movement while preserving a short opacity
transition.

## API reference

### ThreeDButton

| Prop       | Type                                      | Default    | Description                    |
| ---------- | ----------------------------------------- | ---------- | ------------------------------ |
| `children` | `ReactNode`                               | —          | Button content.                |
| `variant`  | `"solid" \| "secondary" \| "destructive"` | `"solid"`  | Visual treatment.              |
| `size`     | `"sm" \| "md" \| "lg" \| "icon"`          | `"md"`     | Button dimensions and spacing. |
| `type`     | `"button" \| "submit" \| "reset"`         | `"button"` | Native button behavior.        |

All other native `button` props are forwarded to the underlying element.
