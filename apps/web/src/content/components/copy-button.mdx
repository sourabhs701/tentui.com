## Features

- Accepts text as a value or a function resolved when the button is clicked.
- Animates idle, success, and error icons before returning to idle.
- Supports custom state icons and copy outcome callbacks.

## Usage

```tsx
import { CopyButton } from "@/components/copy-button";
```

```tsx
<CopyButton size="default" variant="outline" text="npm install tentui">
  Copy command
</CopyButton>
```

Pass a function when the value should be resolved at click time.

```tsx
<CopyButton text={() => window.location.href} aria-label="Copy page URL" />
```

## Accessibility

`CopyButton` preserves the underlying button's keyboard and focus behavior and
uses `Copy` as its default accessible label. Set `aria-label` to describe a more
specific action when needed.

Success and error feedback is visual. Announce the result separately when
confirmation needs to be available to assistive technology.
